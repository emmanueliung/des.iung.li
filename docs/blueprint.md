# **App Name**: Visual Phase

## Core Features:

- Animated Portfolio Display: Showcase graphic design work through animated phase transitions to highlight visual appeal and technical skill.
- AI-Driven Tagging: Automatically tag portfolio items using a generative AI tool to categorize projects by style, technology, and industry for enhanced searchability and SEO.
- Multilingual Content Management: Enable content translation into French, Spanish, and English, with a focus on European and US visibility.
- Contact Form: Implement a contact form to facilitate project inquiries, ensuring submissions are directed to the designer's preferred email.
- SEO Optimization: Ensure the website is optimized for search engines to increase visibility, especially in Europe and the United States.

## Style Guidelines:

- Primary color: A deep, saturated blue (#3F51B5) evoking trust, professionalism, and technological expertise.
- Background color: A light, desaturated blue (#E8EAF6), providing a clean and modern backdrop to highlight the portfolio items.
- Accent color: A vibrant, contrasting orange (#FF9800) to draw attention to calls to action and important elements.
- Headline font: 'Poppins', sans-serif, for a modern and geometric feel.
- Body font: 'PT Sans', sans-serif, for readability and a touch of warmth.
- Implement subtle phase animations for transitions between portfolio items to capture the user's attention.
- Emphasize a visual-centric layout, ensuring minimal text and maximum focus on the showcased design projects.
